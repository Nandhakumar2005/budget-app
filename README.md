# 💰 Budget Tracker App

A simple React.js app to track income and expenses in real time.

---

## 🚀 Features

- Add income and expenses
- Real-time balance update
- Transaction list view
- Delete transactions
- Responsive design

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript
- CSS

---

## ⚙️ Setup

```bash
npm install
npm run dev

🚀 Build & Deploy
npm run build
npm run deploy
